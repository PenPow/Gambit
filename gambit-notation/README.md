<h1 align="center">gambit-notation</h1>
<p align="center">
  Parsers for various forms of chess notations.
</p>

  <a href="https://crates.io/crates/gambit-notation"><img src="https://img.shields.io/crates/v/gambit-notation.svg" alt="crates.io" /></a>

`gambit-notation` provides various parsers to help parse chess notation.

## Features

| Feature | Description                          |
|---------|--------------------------------------|
| `fen`   | Parsers for Forsyth-Edwards Notation |
| `lan`   | Parsers for long algebraic notation  |

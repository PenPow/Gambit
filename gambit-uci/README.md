<h1 align="center">gambit-uci</h1>
<p align="center">
  A UCI-compatible parser for the gambit-protocol communication standard
</p>
<p align="center">
  <a href="https://crates.io/crates/gambit-uci"><img src="https://img.shields.io/crates/v/gambit-uci.svg" alt="crates.io" /></a>
  <a href="https://docs.rs/gambit-uci"><img src="https://docs.rs/gambit-uci/badge.svg" alt="docs.rs" /></a>
  <img src="https://img.shields.io/badge/license-MIT%20OR%20Apache--2.0-blue.svg" alt="License" />
</p>

`gambit-uci` implements a parser and formatter to handle events and commands defined in the `gambit-protocol` crate, allowing
support for UCI chess interfaces.
